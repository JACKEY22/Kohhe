# 카페를 소개하는 게시판 [https://kohhee.shop]

### app
- account : 사용자 계정
- profile : 사용자의 프로필 
- article : 사용자가 작성하는 게시글
- comment : 사용자가 게시글에 작성하는 댓글 
- subscription : 사용자가 원하는 계정과 카페 계정을 팔로우 
- shop : 카페 계정
